# Handwritten Text Recognition System
In this project, we are going to use the Tesseract OCR technique to recognize the handwritten text from the images. The user is prompted to upload an image, which is then processed to recognize the text from the image.

## Introduction
This is a simple yet effective handwritten text recognition system. It utilizes image processing techniques in combination with Optical Character Recognition (OCR) technology to recognize handwritten text from images. The system can recognize handwritten text from images and translate it into digital text. The system can help us understand how we can use simple artificial intelligence tools to solve real-world problems such as manual text input and handwritten data digitization.

## Problem Statement
It is very difficult to read handwritten text from images. In the real world, there are several situations in which handwritten data has to be processed.

### Proposed Solution
The solution for this project will utilize Tesseract OCR. The user can upload the image, which will then undergo some processing. Finally, the text will be extracted from the image and displayed.
